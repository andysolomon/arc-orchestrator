# Plugin Surface Sync Slash Commands

Use these only in repositories that contain orchestrator plugin code. Copy one command into Claude Code TUI.

```text
/arc-orchestrator:orchestrate review the orchestrator plugin surfaces for drift. Compare Claude Code skills, worker agents, README guidance, tests, and shared prompt factory wording. Identify wording or behavior that should be centralized instead of duplicated. Read-only. Do not edit files. Label the run plugin-surface-sync.
```
